# BC Family Trip 2026 🌲

**19 July – 20 August · Vancouver · Deep Cove · Sea-to-Sky · Vancouver Island · Sunshine Coast**

A 33-day family trip itinerary through British Columbia, built as a static site for GitHub Pages.

## Pages

| Page | Content |
|---|---|
| [**Overview**](index.html) | Trip at a glance, segment cards, booking priorities |
| [**Itinerary**](itinerary.html) | Day-by-day plan with distances, energy ratings, insider tips |
| [**Logistics**](logistics.html) | Ferry strategy, booking tracker, energy map, distance summary |
| [**Kids Corner**](kids.html) | Beaches, animals, walks, treats, travel games, fun facts |
| [**Appendices**](appendices.html) | History, tides/wildlife, food guide, rainy-day alternatives |

## Key Numbers

- **33 days / 32 nights** across 6 segments
- **~1,300 km** total driving + **4 ferry crossings**
- **3 "watch carefully" days** flagged for logistics risk
- **Car rental:** 22 July – 17 August (27 days)

## Deployment

This is a static HTML/CSS/JS site. To deploy on GitHub Pages:

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Set source to main branch, root directory
4. The site will be available at `https://<username>.github.io/bc-trip-2026/`

## Design

- **Palette:** BC pastels — ocean blue, forest sage, sand, driftwood, sunset coral
- **Typography:** Playfair Display (headings) + DM Sans (body)
- **Images:** Unsplash (landscape/nature) — no personal photos
- **No personal information** is stored or displayed on the site
